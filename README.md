### 发布方式
```js
grunt dev          编译
grunt commit:dev   提交到测试环境
grunt test         编译beta
grunt commit:test  提交beta


grunt release:css  //本地less编译调试	

grunt build        //生成线上代码
grunt commit:pro   发布


grunt test	   //生成测试环境包
grunt clear	   //清除文件

grunt upload	
grunt commit//上传测试环境


grunt require	//合并页面级文件
grunt check	//检查代码规范
grunt jsmin	//压缩js 	
grunt cssmin	//压缩css
```
